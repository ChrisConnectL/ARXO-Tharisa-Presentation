Bulk Liquid Diesel (Zone 9C) and Chrome FOB Logistics

To present transparent logistics cost models and operating parameters for review and discussion.

How To Read This Proposal

Two independent logistics models are presented: Leg 1 deals with bulk liquid diesel delivery into the inland / mine corridor, while Leg 2 deals with chrome logistics back to Durban and FOB readiness. Each leg should be reviewed separately.

Fuel pricing is shown as an input reference only: The diesel price is used to build an illustrative delivered-cost model and is not presented as a fuel-trading margin or resale proposal.

No comparison is made to ARXO/Tharisa's procurement model: Connect Logistics does not have visibility of ARXO/Tharisa's current fuel procurement structure or price per litre, so the presentation must avoid direct comparison.

All figures are illustrative and parameter-based: The model depends on stated assumptions such as volume, route, cycle time, equipment configuration, utilisation and transport cost.

The intent is alignment and validation: The purpose is to create a shared understanding before any Teams discussion, POC structure or commercial commitment.

Fuel pricing is shown as a dated reference input only: The diesel reference used in this model is based on the published position as at 05/05/2026, before the diesel price increase effective 06/05/2026. It is therefore used only as a reference point for the costing simulation, not as a fixed or current fuel offer.

This is a costing simulation, not a fuel-trading proposal or price-comparison document.

The corridor is viewed as two distinct logistics legs: diesel moving inland and chrome moving back to Durban. The operating corridor is linked, but the commercial models remain separate.

Bulk Liquid Diesel: Key Parameters & Guardrails

Model Assumptions

  • Delivery area: Zone 9C / inland mine corridor.
  • Volume basis: ±1.6 million litres/month.
  • Equipment configuration: MTS tanks on flatbed superlink trailers.
  • Operational inputs: route, cycle time, delivery windows, site access and utilisation.
  • Scope boundary: fuel input and logistics cost are shown separately.

Guardrails & Validation

  • The diesel price reference is date-specific: the model uses the 05/05/2026 published price position before the 06/05/2026 increase and should be updated if the model is refreshed.
  • Fuel price is a variable input: it is not controlled by the logistics model.
  • Logistics cost is the controllable component: route, cycle time, fleet configuration and utilisation drive this element.
  • No ARXO/Tharisa price comparison is made: the model does not benchmark against ARXO/Tharisa's current delivered price.
  • Assumptions must be validated using measured operating data.
